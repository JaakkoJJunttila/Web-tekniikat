Git & GitHub - Getting Started

1. Install Git (https://git-scm.com/downloads).
2. Create an account in GitHub (https://github.com/).
3. Create a new folder Web-tekniikat in VS Code.
4. Create a local repository of this folder.
5. Publish your repository in a public repository in GitHub.
6. Create a simple HTML, CSS, JS page in your repo and push changes to your repository.
